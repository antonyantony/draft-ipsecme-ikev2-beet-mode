Command Line Usage

Formatted text,HTML and xml versions of the draft can be built using make.

$ make
ls -l draft/*-latest*

Command line usage requires that you have the necessary software installed. A local copy of emcas, which is used by the make. You are free to use an editor of your choice. Alternatively "make DOCKER=yes"
